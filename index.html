Prettydateintl

jQuery plugin to make the date pretty. i18n compilant

Download this project as a .zip file Download this project as a tar.gz file

Humanize your dates

Instead of ugly dates such as Monday, April, 1st 2013 or 2012-09-18 20:00:00 display relative dates such as just now, yesterday or 2 years ago.

Simply apply the dateme() method on the elements you would like to be changed.

$('.date').dateme();

Internationalize (i18n) your dates

What if you want your dates to be displayed in French, Korean or Klingon ? Just set the language code of your choice such as lang:'de' or lang:'de_LU'.

$('.date').dateme({lang:'fr'})

You can use featured languages packs or create your own.
